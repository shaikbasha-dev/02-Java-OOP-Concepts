# Source Code

## Overview

The **Source-Code** directory contains a well-structured collection of Java programs covering the complete journey of **Object-Oriented Programming (OOP)** concepts in Java. Each program focuses on a specific concept and is designed to provide a clear understanding through simple, beginner-friendly examples.

The programs follow a progressive learning approach, starting from object creation and constructors, then moving through inheritance, polymorphism, abstraction, interfaces, and wrapper classes. Every source file is written with professional coding standards, meaningful comments, clean formatting, and concept-oriented examples suitable for academic learning, revision, and technical interview preparation.

---

# Learning Objectives

This collection helps learners understand:

- Creating and using Java objects
- Constructors and constructor overloading
- Encapsulation
- Types of inheritance
- Method overriding
- Compile-time polymorphism
- Runtime polymorphism
- Dynamic Method Dispatch
- Abstraction
- Interfaces
- Abstract, Default, and Static methods in interfaces
- Wrapper Classes
- Java programming best practices
